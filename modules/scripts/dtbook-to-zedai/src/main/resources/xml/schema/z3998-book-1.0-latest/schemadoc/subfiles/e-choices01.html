Allowed parents

xforms:choices, xforms:select and xforms:select1

Allowed children

This element must not contain text.

This element may contain the following children: xforms:choices, xforms:item, xforms:itemset and xforms:label

Content model

optional attributes: @class, @its:dir, @its:translate, @role, @xml:base, @xml:id, @xml:lang and @xml:space
an optional xforms:label
one or more of
any of the 3 elements: xforms:choices, xforms:item and xforms:itemset

Namespace

http://www.w3.org/2002/xforms/
