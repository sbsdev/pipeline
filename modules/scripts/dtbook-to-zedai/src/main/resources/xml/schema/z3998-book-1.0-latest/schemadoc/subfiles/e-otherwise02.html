Container for a content fragment to be selected if none of the preceding sel:when (Phrase) conditions evaluated to true.

Usage Example

        
<sel:select>
    <sel:when expr="…">
        …
    </sel:when>    
    <sel:otherwise>
        …
    </sel:otherwise>
</sel:select>

Allowed parents

sel:select

Allowed children

  • The sel:otherwise element must not have sel:select descendants.

Variants

This element exists in 2 variants. The other variant is the Block variant.
