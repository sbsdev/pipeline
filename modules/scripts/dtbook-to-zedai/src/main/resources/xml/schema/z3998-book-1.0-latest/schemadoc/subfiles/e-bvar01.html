Allowed parents

m:apply, m:bind, m:lambda, m:list, m:matrix, m:matrixrow, m:set and m:vector

Allowed children

This element must not contain text.

This element may contain the following children: m:ci, m:degree and m:semantics

Content model

a mix of
• any of the 2 elements: m:ci and m:semantics
• an optional m:degree
end of mix

Namespace
