Allowed parents

m:bvar and m:semantics

Allowed children

This element must not contain text.

This element may contain the following children: m:annotation, m:annotation-xml, m:ci and m:semantics

Content model

optional attributes: @cd, @definitionURL, @encoding, @name, @xml:base and @xml:space
any of the 2 elements: m:ci and m:semantics
zero or more of
any of the 2 elements: m:annotation and m:annotation-xml

Variants

This element exists in 3 variants. The other 2 variants are:

  • The UNDEFINED variant of the m:semantics element
  • The UNDEFINED variant of the m:semantics element
