Allowed parents

xforms:choices, xforms:select and xforms:select1

Allowed children

This element must not contain text.

This element must contain the following children: xforms:label

This element may contain the following children: xforms:copy and xforms:value

Content model

optional attributes: @bind, @class, @its:dir, @its:translate, @model, @nodeset, @role, @xml:base, @xml:id, @xml:lang and @xml:space
xforms:label
any of the 2 elements: xforms:copy and xforms:value
