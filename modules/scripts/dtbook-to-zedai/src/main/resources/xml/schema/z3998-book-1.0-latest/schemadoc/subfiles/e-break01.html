The break element

Controls the pausing or other prosodic boundaries between tokens.

Refer to SSML 1.1 for further information.

Usage Example

<p>… </p>
<transition />
<ssml:break strength="strong" />        
<p>…</p>
    
<p>…</p>
<transition />
<ssml:break time="2s" />
<p>…</p>
