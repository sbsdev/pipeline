The tr element represents a single table row of data.

Usage Example

<table>
    <tr>
        <th colspan="2">Inches</th>
        <th>Lines</th>
    </tr>
    <tr>
        <th>Length (as far as complete)</th>
        <td>17</td>
        <td>6</td>
    </tr>
    <tr>
        <th>Depth of each ramus</th>
        <td>4</td>
        <td>9</td>
    </tr>
</table>

Allowed parents

table, tbody, tfoot and thead

Allowed children

This element must not contain text.

This element may contain the following children: pagebreak, td and th

Content model

optional attributes: @about, @by, @class, @content, @continuation, @datatype, @desc, @its:dir, @its:translate, @property, @rel, @resource, @rev, @role, @sel:selid, @ssml:onlangfailure, @typeof, @xml:base, @xml:id, @xml:lang and @xml:space
a mix of
• one or more of
any of the 2 elements: td and th
• zero or more of pagebreak
end of mix
