xprocspec - XProc testing tool

This is a tool for testing XProc scripts. Its grammar is inspired by XSpec and is implemented in XProc.

XProc

Using Calabash (calabash 1.0.10-95 or newer is recommended), provide the xprocspec test on the "source" input port, and point the "result", "html" and "junit" report ports to a location of your choice.

NOTE: In Windows, you must also provide a file URI to the "temp-dir" option. It defaults to "file:/tmp/" which works in Linux and OSX.

Maven

If you use the Maven plugin xproc-maven-plugin, you can do automated testing of your XProc projects! Read more on xproc-maven-plugin project homepage.

Get started now!

wget http://daisy.github.io/xprocspec/www/hello-world.zip
unzip hello-world.zip
cd hello-world
mvn test

This will download a "hello world" template project and run a sample xprocspec test (as well as a xspec test, using xspec-maven-plugin).
